Drag & Drop in HTML5

Events
  • dragstart - dragging starts, fired on dragged element
  • dragenter - mouse enters an element 1) 2)
  • dragover - mouse moves over an element 1)
  • dragleave - mouse leaves an element 2)
  • drag - similar to dragstart
  • drop - drop successful
  • dragend - fired on dragged element whether successful or not

1) Indicate whether drop is allowed
2) Highlight drop area

What can be dragged & dropped?
Files, images, HTML document nodes, ...

Drag & Drop in HTML5


<img id.. draggable="true" ondragstart="drag(event)" />

<div id.. ondrop="drop(event)" ondragover="allowDrop(event)"></div>
                

function drag(ev) {
    ev.dataTransfer.setData("text", ev.target.id);
}

function allowDrop(ev) {
    ev.preventDefault(); 
    ev.dataTransfer.dropEffect = 'move';
    // ev.dataTransfer.getData('text') can't be used here
}

function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData('text');
    ev.target.appendChild(document.getElementById(data));
}

Advanced Features

  • A placeholder where the element is placed if dropped

draggable

Link Function: 

angular.element(element).attr('draggable', 'true');

element.bind('dragstart', function () {
    var storyId = angular.element(element).attr('story-id');
    localStorage.setItem('draggedStoryId', storyId);
    $rootScope.$emit('storyDraggingStart');
});

element.bind('dragend', function () {
    $rootScope.$emit('storyDraggingEnd');
});

dropable

Link Function: 

var id = attrs.id; // generate id, if no id exists

element.bind('drop', function (e) {
    e.stopPropagation(); e.preventDefault();
    
    var src = getStory(storyId);
    var dest = document.getElementById(id);
    var storyId = localStorage.getItem('draggedStoryId');
    
    $rootScope.$broadcast('story-dragged', src, dest, storyId);
});

The solution with a placeholder looks slightly more complicated: draggableStory.js

Compromises & Lessons Learned

  • HTML5 Storage You might want to use localStorage instead of the data transfer of the event, because it just works
  • Where did I drop it?Search for the first "dropable" parent of ev.target with $(e.target).closest('[dropable]')
  • Look&FeelAdd a placeholder to improve the look&feel (and to make your life more complicated)
  • Custom EventsUse custom events with parameters: $rootScope.$broadcast(..)

// spec.js
describe('Protractor Demo App', function() {
  it('should add one and two', function() {
    browser.get('http://juliemr.github.io/protractor-demo/');
    element(by.model('first')).sendKeys(1);
    element(by.model('second')).sendKeys(2);

    element(by.id('gobutton')).click();

    expect(element(by.binding('latest')).getText()).
        toEqual('5'); // This is wrong!

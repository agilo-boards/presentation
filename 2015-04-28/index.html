Drag & Drop

Directives

  • draggable element to drag
  • dropable area(s) to drop the element into

Advanced Features

  • Placeholder where the element is placed if dropped

draggable

Link Function: 

angular.element(element).attr('draggable', 'true');

element.bind('dragstart', function () {
    var storyId = angular.element(element).attr('story-id');
    localStorage.setItem('draggedStoryId', storyId);
    $rootScope.$emit('storyDraggingStart');
});

element.bind('dragend', function () {
    $rootScope.$emit('storyDraggingEnd');
});

dropable

Link Function: 

var id = attrs.id; // generate id, if no id exists

element.bind('drop', function (e) {
    e.stopPropagation(); e.preventDefault();
    
    var src = getStory(storyId);
    var dest = document.getElementById(id);
    var storyId = localStorage.getItem('draggedStoryId');
    
    $rootScope.$broadcast('story-dragged', src, dest, storyId);
});

Mit Placeholder sieht es etwas komplizierter aus. draggableStory.js

Notes

  • localStorage statt Speichern auf dem Event selbst
  • Noch nicht ausgereift
Protractor
UI Testing with Selenium

Protractor

  • e2e test framework
  • built on top of WebdriverJS
  • for angularjs apps

Grunt plugin

  • grunt-protractor-runner

Example of a test


// spec.js
describe('Protractor Demo App', function() {
  it('should add one and two', function() {
    browser.get('http://juliemr.github.io/protractor-demo/');
    element(by.model('first')).sendKeys(1);
    element(by.model('second')).sendKeys(2);

    element(by.id('gobutton')).click();

    expect(element(by.binding('latest')).getText()).
        toEqual('5'); // This is wrong!
  });
});

configuration file


//conf.js
exports.config = {
    seleniumServerJar: './node_modules/protractor/selenium/selenium-server-standalone-2.42.2.jar',
    seleniumPort: null,
    chromeDriver: './node_modules/protractor/selenium/chromedriver',
    seleniumArgs: [],

    capabilities: {
        'browserName': 'chrome'
    },
    specs: ['test/spec/end2End/*.js'],
  
    jasmineNodeOpts: {
        showColors: true
    }
};

global variables

  • browser - browser.get(url)
